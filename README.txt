== Description ==
MinMax Quantities for WooCommerce help you to restrict minimum and maximum quantity for product. You can set the restriction globally, as well as individual product including variations. You can also set amount and quantity limits for cart and order.

== Installation ==

The Plugin installation process in WordPress is very easy and convenient. Simply follow the steps below to get started.

1. Upload `minmax-quantities-for-woocommerce` plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set the quantity restriction from WooCommerce -> Min Max Quantity
